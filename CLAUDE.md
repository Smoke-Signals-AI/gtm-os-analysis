# Platform & API reference

Read `PLATFORM.md` (repo root) before writing or changing any external-API
integration. It documents the full Smoke Signals AI product landscape (ContentOS,
SocialOS, DealOS, GTM OS, Client Management Portal), every external API the fleet
uses (HubSpot, Anthropic, Anysite, Unipile, Parallel.ai, ElevenLabs, Deepgram,
Stripe, ...), and the integration best practices we've learned across projects.
Update the relevant section in the same PR when an integration's behavior changes
or a new gotcha is discovered.
